### 6.1.1 IK基础概念

**逆向运动学（Inverse Kinematics）**是骨骼动画中的重要技术，它允许通过指定末端效应器（如手、脚）的目标位置，自动计算出中间关节的旋转角度。

```cpp
// 正向运动学示例：从根关节开始计算末端位置
```

### 6.1.2 Two Bone IK算法

**Two Bone IK**是虚幻引擎中最常用的IK算法，适用于处理手臂、腿部等两段骨骼的IK问题。

#### 算法原理

```cpp
// Two Bone IK核心算法
bool SolveTwoBoneIK(
    FVector RootPos, FVector JointPos, FVector EndPos, // 骨骼位置
    FVector TargetPos, FVector PoleVector,           // 目标位置和极向量
    float UpperLimbLength, float LowerLimbLength,     // 骨骼长度
    FVector& OutNewJointPos, FVector& OutNewEndPos   // 输出结果
) {
    // 1. 计算当前骨骼链长度和到目标的距离
    float CurrentLength = (EndPos - RootPos).Size();
    float TargetDistance = (TargetPos - RootPos).Size();
    
    // 2. 检查是否在可到达范围内
    float MaxReach = UpperLimbLength + LowerLimbLength;
    if (TargetDistance >= MaxReach) {
        // 目标超出范围，直接拉伸
        FVector Direction = (TargetPos - RootPos).GetSafeNormal();
        OutNewJointPos = RootPos + Direction * UpperLimbLength;
        OutNewEndPos = TargetPos;
        return true;
    }
    
    // 3. 使用余弦定理计算关节角度
    float CosAngle = (UpperLimbLength * UpperLimbLength + 
                     LowerLimbLength * LowerLimbLength - 
                     TargetDistance * TargetDistance) / 
                    (2 * UpperLimbLength * LowerLimbLength);
    
    CosAngle = FMath::Clamp(CosAngle, -1.0f, 1.0f);
    float Angle = FMath::Acos(CosAngle);

    // ... 详细实现
    
    return true;
}
```

### 6.1.3 FABRIK算法

**FABRIK（Forward And Backward Reaching Inverse Kinematics）**是一种高效的IK算法，适用于多骨骼链的IK求解。

#### 算法步骤

```cpp
// FABRIK算法实现
void SolveFABRIK(
    TArray<FVector>& BonePositions,
    const TArray<float>& BoneLengths,
    const FVector& TargetPosition,
    int32 MaxIterations = 10,
    float Tolerance = 0.01f
) {
    int32 NumBones = BonePositions.Num();
    
    for (int32 Iteration = 0; Iteration < MaxIterations; ++Iteration) {
        // 前向传递：从末端到根节点
        BonePositions[NumBones - 1] = TargetPosition;
        for (int32 i = NumBones - 2; i >= 0; --i) {
            FVector Direction = (BonePositions[i] - BonePositions[i + 1]).GetSafeNormal();
            BonePositions[i] = BonePositions[i + 1] + Direction * BoneLengths[i];
        }
        
        // 后向传递：从根节点到末端
        // 根节点位置保持不变
        for (int32 i = 1; i < NumBones; ++i) {
            FVector Direction = (BonePositions[i] - BonePositions[i - 1]).GetSafeNormal();
            BonePositions[i] = BonePositions[i - 1] + Direction * BoneLengths[i - 1];
        }
        
        // 检查收敛
        float Error = (BonePositions.Last() - TargetPosition).Size();
        if (Error < Tolerance) break;
    }
}
```

### 6.2.2 Ragdoll物理系统

**Ragdoll**系统允许角色在死亡或失去意识时进入物理模拟状态，产生真实的摔倒效果。

#### Ragdoll激活流程

```cpp
// 激活Ragdoll物理模拟
void ACharacter::EnableRagdoll() {
    if (USkeletalMeshComponent* SkeletalMesh = GetMesh()) {
        // 1. 禁用动画更新
        SkeletalMesh->SetSimulatePhysics(true);
        SkeletalMesh->SetCollisionEnabled(ECollisionEnabled::PhysicsOnly);
        
        // 2. 设置物理参数
        SkeletalMesh->SetAllBodiesBelowSimulatePhysics("spine_01", true);
        SkeletalMesh->SetEnableGravity(true);
        
        // 3. 应用初始冲量
        SkeletalMesh->AddImpulse(DeathImpulse, NAME_None, true);
    }
}
```

#### 创建自定义节点

```cpp
// 自定义动画节点基类
UCLASS(BlueprintType)
class UCustomAnimNode : public UAnimNode_Base {
    GENERATED_BODY()
    
public:
    // 节点输入引脚
    UPROPERTY(EditAnywhere, BlueprintReadWrite, Category = "Input")
    FPoseLink InputPose;
    
    // 自定义参数
    UPROPERTY(EditAnywhere, BlueprintReadWrite, Category = "Settings")
    float CustomParameter;
    
    // 重写评估函数
    virtual void Evaluate_AnyThread(FPoseContext& Output) override {
        // 输入姿势评估
        InputPose.Evaluate(Output);
        
        // 自定义逻辑处理
        ApplyCustomAnimation(Output);
    }
    
    // 自定义动画处理函数
    void ApplyCustomAnimation(FPoseContext& Output) {
        // 实现自定义动画逻辑
        for (int32 BoneIndex = 0; BoneIndex < Output.Pose.GetNumBones(); ++BoneIndex) {
            FTransform& BoneTransform = Output.Pose[BoneIndex];
            // 应用自定义变换
            BoneTransform.SetRotation(BoneTransform.GetRotation() * 
                FQuat(FVector::UpVector, CustomParameter));
        }
    }
};
```

```cpp
UCLASS()
class UScaleAnimationModifier : public UAnimModifier {
    GENERATED_BODY()
    
public:
    UPROPERTY(EditAnywhere, BlueprintReadWrite)
    float ScaleFactor = 1.0f;
    
    virtual void OnApply_Implementation(UAnimSequence* Animation) override {
        // 缩放动画时长
        float OriginalLength = Animation->SequenceLength;
        Animation->SequenceLength = OriginalLength * ScaleFactor;
        
        // 调整关键帧时间
        for (auto& Track : Animation->GetRawAnimationData()) {
            for (auto& Key : Track.PosKeys) {
                Key.Time *= ScaleFactor;
            }
        }
    }
};
```

- **中端手机**：30-50根骨骼  
- **低端手机**：15-30根骨骼

#### LOD系统实现

```cpp
// 动画LOD系统
class UAnimationLODSystem : public UObject {
public:
    // 根据设备性能选择LOD级别
    int32 GetOptimalLODLevel() const {
        // 基于设备性能评分选择LOD
        float PerformanceScore = CalculateDevicePerformance();
        
        if (PerformanceScore > 0.8f) return 0; // 最高质量
        else if (PerformanceScore > 0.5f) return 1; // 中等质量
        else return 2; // 最低质量
    }
    
    // 应用LOD设置
    void ApplyLODSettings(USkeletalMeshComponent* SkeletalMesh, int32 LODLevel) {
        switch (LODLevel) {
            case 0: // 高质量：完整骨骼
                SkeletalMesh->SetUpdateAnimationInEditor(true);
                break;
            case 1: // 中等质量：简化骨骼
                SkeletalMesh->SetUpdateAnimationInEditor(false);
                ApplyBoneReduction(0.3f); // 减少30%骨骼
                break;
            case 2: // 低质量：最小骨骼
                SkeletalMesh->SetUpdateAnimationInEditor(false);
                ApplyBoneReduction(0.6f); // 减少60%骨骼
                break;
        }
    }
};
```
