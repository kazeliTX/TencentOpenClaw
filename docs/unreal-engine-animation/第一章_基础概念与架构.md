# 第一章：基础概念与架构

## 1.1 虚幻引擎动画系统概述

虚幻引擎的动画系统是一个高度优化的实时动画解决方案，它提供了从角色动画到环境动画的完整工具链。系统基于组件化架构，支持从简单的关键帧动画到复杂的物理驱动动画。

### 核心组件架构

**SkeletalMeshComponent**
- 功能：骨骼网格渲染和动画播放的核心组件
- 属性：骨骼变换、材质、碰撞体配置
- 方法：播放动画、混合动画、事件处理

**AnimInstance**
- 功能：动画逻辑控制器，管理状态机和混合逻辑
- 特性：蓝图可编程、多线程安全更新
- 应用：角色动画、物理动画、逆向运动学计算

**AnimationSequence**
- 功能：基础动画片段，存储关键帧数据
- 格式：支持ACL、Bitwise等压缩算法
- 应用：行走、奔跑、跳跃等基础动作

### 动画管线流程

1. **资源导入阶段**
   - FBX/Maya等格式导入
   - 骨骼绑定和权重分配
   - 动画数据压缩和优化

2. **运行时处理阶段**
   - 动画序列采样
   - 骨骼变换计算
   - 混合和叠加处理
   - 最终渲染输出

## 1.2 骨骼系统基础

### 骨骼层级结构

```
Root
├── Pelvis
│   ├── Spine
│   │   ├── Neck
│   │   │   └── Head
│   │   └── Shoulder_L
│   │       └── Arm_L
│   └── Thigh_L
└── Thigh_R
```

### 骨骼变换计算

**局部空间 vs 世界空间**
- 局部空间：骨骼相对于父骨骼的变换
- 世界空间：骨骼在场景中的最终位置
- 绑定姿势：骨骼的初始参考姿势

**骨骼矩阵计算**
```cpp
// 局部到组件空间变换
FTransform LocalToComponent = BoneTransform * ParentTransform;

// 组件到世界空间变换  
FTransform ComponentToWorld = SkeletalMeshComp->GetComponentTransform();

// 最终世界空间变换
FTransform WorldTransform = LocalToComponent * ComponentToWorld;
```

## 1.3 动画资源类型

### Animation Sequence
- **功能**：基础动画片段
- **关键帧数据**：时间轴上的动画采样
- **压缩格式**：ACL、Bitwise等压缩算法
- **曲线数据**：浮点曲线、变换曲线

### Blend Space
- **功能**：参数驱动的动画混合
- **类型**：1D、2D、3D混合空间
- **应用**：移动、转向等连续动画
- **配置**：混合权重和过渡曲线

### Montage
- **功能**：动画片段组合
- **特性**：支持分层和混合
- **应用**：攻击、交互等复杂动作
- **事件**：动画通知和状态机集成

### Pose Asset
- **功能**：静态姿势资源
- **应用**：IK目标、参考姿势
- **优势**：内存占用小，加载快速
