### 7.1.1 角色移动系统设计

#### 移动状态机架构

**基础状态设计**：
```cpp
// 角色移动状态枚举
UENUM(BlueprintType)
enum class ECharacterMovementState : uint8
{
    Idle        UMETA(DisplayName = "待机"),
    Walk        UMETA(DisplayName = "行走"),
    Run         UMETA(DisplayName = "奔跑"),
    Sprint      UMETA(DisplayName = "冲刺"),
    Jump        UMETA(DisplayName = "跳跃"),
    Fall        UMETA(DisplayName = "下落"),
    Land        UMETA(DisplayName = "落地"),
    Crouch      UMETA(DisplayName = "蹲下"),
    Prone       UMETA(DisplayName = "匍匐")
};
```

**状态转换条件**：
- **Idle → Walk**：输入移动向量且速度 > 0
- **Walk → Run**：移动速度超过阈值且按住加速键
- **Run → Sprint**：体力充足且按住冲刺键
- **任意 → Jump**：按下跳跃键且在地面
- **Jump → Fall**：垂直速度 < 0
- **Fall → Land**：检测到地面碰撞

```cpp
void ACharacterBase::TakeDamage(float Damage, EHitReactionType HitType, FVector HitDirection)
{
    // 根据伤害类型选择受击动画
    if (auto* ReactionData = HitReactionMap.Find(HitType))
    {
        // 播放受击蒙太奇
        PlayAnimMontage(ReactionData->ReactionMontage);
        
        // 设置硬直状态
        if (ReactionData->StunDuration > 0)
        {
            SetStunned(ReactionData->StunDuration);
        }
        
        // 中断当前攻击
        if (ReactionData->bInterruptAttack && IsAttacking())
        {
            InterruptCurrentAttack();
        }
    }
}
```

```cpp
// 波浪数据
struct FWaveParameters
{
    float Amplitude;       // 波幅
    float Wavelength;      // 波长
    float Speed;          // 波速
    FVector2D Direction;  // 波向
    float Steepness;      // 波陡
};

// 多波浪叠加
struct FMultiWaveSystem
{
    TArray<FWaveParameters> Waves;
    float Time;           // 时间累积
    
    // 计算水面高度
    float CalculateWaterHeight(const FVector2D& Position) const
    {
        float Height = 0.0f;
        for (const auto& Wave : Waves)
        {
            float DotProduct = FVector2D::DotProduct(Position, Wave.Direction);
            float Phase = (DotProduct / Wave.Wavelength + Time * Wave.Speed) * 2 * PI;
            Height += Wave.Amplitude * FMath::Sin(Phase);
        }
        return Height;
    }
};
```
