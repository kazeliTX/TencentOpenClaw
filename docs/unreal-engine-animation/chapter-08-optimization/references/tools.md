# 性能分析工具

| 工具 | 用途 | 获取方式 |
|------|------|---------|
| Unreal Insights | 完整性能追踪（推荐）| 随 UE 引擎附带 |
| stat anim | 实时动画统计 | 游戏内控制台 |
| RenderDoc | GPU 帧分析 | https://renderdoc.org |
| Visual Studio Profiler | CPU 热点（C++ 代码）| Visual Studio 内置 |
| Superluminal | 轻量级 CPU Profiler | https://superluminal.eu |
| Animation Insights（UE5）| 可视化动画节点耗时 | UE 编辑器内置 Window |
