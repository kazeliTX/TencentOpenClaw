# 10.7 浅水模拟

> **难度**：⭐⭐⭐⭐⭐

```
浅水模拟（Shallow Water Simulation）：
  适合：水坑/浅河/洪水漫延
  
  UE5 实现方案：
    Niagara Grid2D_Shallow_Water（推荐）
    分辨率：64×64（实时交互）～ 128×128（精细）

关键物理参数：
  Gravity：重力加速度（影响波速）
  H0（静止水深）：平均水深（m）
  波速 = sqrt(g × H0)
    H0=1m → 波速 ≈ 3.1 m/s（慢）
    H0=5m → 波速 ≈ 7.0 m/s（快）

与场景物体交互：
  在场景中放置障碍物 → 对应格子的流速被阻断
  → 水流绕过障碍物（需要在 Niagara 中实现遮挡 mask）

洪水漫延效果：
  动态修改 H0 参数（随时间增大）
  → 水位上升，向低处流淌
  监测水面与地形高度 → 触发浮力/淹没效果
```
