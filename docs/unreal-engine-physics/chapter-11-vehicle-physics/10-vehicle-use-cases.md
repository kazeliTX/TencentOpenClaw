# 11.10 应用案例（赛车/坦克/摩托）

> **难度**：⭐⭐⭐⭐☆

```
1. 赛车游戏
   WheelSetup：4轮，前轮转向角 35°
   发动机：高 MaxRPM(9000), 峰值扭矩曲线集中在中高转速
   变速箱：6档，ChangeUpRPM = 7500
   空气动力学：DownforceCoeff 随速度增大（弯道稳定）
   漂移：后驱 + 手刹锁后轮
   音效：转速曲线 → 引擎音效 Pitch

2. 坦克（履带车）
   Chaos Vehicle 模拟轮式底盘（不支持真实履带）
   用多个轮子模拟履带（每侧 4~6 个）
   通过对左右轮施加不同扭矩实现转向（差速转向）
   履带视觉：用骨骼动画驱动履带板
   
3. 摩托车
   2个轮（前后）+ 平衡控制器
   Chaos Vehicle 原生不支持 2 轮平衡
   需要自写平衡 PID：
     检测车体倾斜角 → 施加扭矩修正
     低速时辅助平衡（像自行车辅助轮）
   
4. APC/装甲车（越野）
   4轮/6轮驱动（AWD）
   SuspensionMaxDrop = 30cm（高通过性）
   TorqueCurve 低转速扭矩大（越野爬坡）
   AngularDamping 高（不容易翻车）
```
