# 6.6 权重绘制（固定/最大距离）

> **难度**：⭐⭐⭐⭐☆

## 一、权重图类型详解

```
Chaos Cloth 的权重图（Paint Maps）：

1. Max Distance（最大移动距离）
   最重要的权重图
   0.0  → 顶点完全固定（焊接到骨骼）
   大值 → 顶点可以自由移动（单位：cm）
   
   典型裙摆分布：
     腰带区域   0.0（完全固定）
     腰以下5cm  0~5 cm（缓慢过渡）
     大腿区域   5~20 cm（适度自由）
     膝盖区域   20~40 cm
     裙摆边缘   40~80 cm（最大自由）

2. Backstop（背部止动）
   防止布料穿透角色身体
   由两个参数组成：
     Backstop Distance：止动球中心到粒子的距离
     Backstop Radius  ：止动球半径
   
   实际效果：粒子不能进入以 (Bone + Distance) 为圆心、
             Radius 为半径的球体内
   
   角色穿裙摆时：
     大腿外侧 Backstop Distance ≈ 腿部半径
     这样裙摆不会穿进大腿里

3. Tether Stiffness（系绳刚度）
   控制顶点被系绳拉回的力度
   高值 → 顶点不会偏离基础姿势太远（保持形状）
   低值 → 顶点可以大幅偏移（飘逸感）

4. Animation Weight
   局部动画/物理混合权重
   0.0 → 完全物理
   1.0 → 完全跟随骨骼动画（等同于固定）
   可以用来做"上半身固定+下半身物理"效果
```

## 二、权重绘制技巧

```
笔刷设置：
  Radius：画笔半径（点击 + 拖拽调整）
  Strength：绘制强度（渐进式绘制）
  Falloff：边缘衰减（=1 硬边，<1 软边）
  
  推荐笔刷操作：
    高 Strength（0.8-1.0）：初步大面积填涂
    低 Strength（0.1-0.3）：细化过渡区域
    
绘制 Max Distance 流程：
  1. Fill All → 0.0（清空，全部固定）
  2. 用梯度填涂：从腰部（0）到裙边（最大值）
  3. 播放模拟，检查是否合理
  4. 迭代调整过渡区域

常见权重问题：
  问题：裙摆运动时有硬折痕
  原因：Max Distance 有硬边界（突变从0到大值）
  修复：用低 Strength 笔刷在边界附近多次涂抹，使过渡平滑
  
  问题：布料完全不动
  原因：所有顶点 Max Distance = 0
  修复：先 Fill All → 设置一个基础值（如20cm）再调整固定区
  
  问题：布料穿透大腿
  原因：Backstop 未配置
  修复：在大腿对应区域绘制 Backstop Distance 和 Backstop Radius
```
