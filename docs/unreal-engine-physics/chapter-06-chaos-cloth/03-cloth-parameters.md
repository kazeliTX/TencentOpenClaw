# 6.3 布料参数详解

> **难度**：⭐⭐⭐⭐⭐

## 一、约束刚度参数

```
参数范围：0.0（完全柔软）~ 1.0（最硬）

Stretch Stiffness（拉伸刚度）：
  0.0 ：布料可以随意拉伸（橡皮感）
  0.5 ：适度弹性（棉布）
  0.8 ：硬挺（丝绸/厚布）
  1.0 ：不可拉伸（帆布/皮革）
  
  低值问题：布料在快速运动时过度拉伸
  高值问题：引入数值刚性，需要更多迭代

Bend Stiffness（弯曲刚度）：
  0.0 ：完全柔软，像湿纸
  0.1~0.3：自然布料感（T恤/薄裙）
  0.5~0.7：厚重布料（大衣/毛毯）
  1.0 ：纸板感（几乎不弯）
  
  注意：弯曲刚度高时边缘容易出现振荡
  修复：增大 Damping 或减少迭代幅度

Area Stiffness（面积保持刚度）：
  控制布料整体面积是否保持不变
  0.0 ：面积可以变化（橡胶膜感）
  1.0 ：严格保持原始面积
  推荐：0.7~0.9（保持形状但允许微小变化）

Shear Stiffness（剪切刚度）：
  防止布料菱形变形（斜线方向拉伸）
  大多数布料：0.5~1.0
  弹性布料（运动服）：0.2~0.4
```

## 二、阻尼与质量

```
Linear Damping（线性阻尼/空气阻力）：
  作用：减慢布料运动，模拟空气摩擦
  0.001 ：真空中的布料（持续飘荡）
  0.01  ：轻薄丝绸
  0.03  ：棉布（推荐起始值）
  0.08  ：厚重毛料
  0.2+  ：水中布料（很黏稠）
  
  太大：布料反应迟钝，像在果冻里
  太小：布料一直飘荡，很久才静止

Angular Damping：
  控制旋转方向的阻尼
  通常比 Linear Damping 稍大（×1.5~2.0）

Uniform Mass（均匀质量，kg）：
  轻薄丝绸：0.003~0.008 kg
  棉布衬衫：0.01~0.03 kg
  厚棉夹克：0.05~0.15 kg
  皮革大衣：0.1~0.3 kg
  帆布帐篷：0.5~2.0 kg
  
  质量小 → 容易被风吹动、飘逸
  质量大 → 沉重感、不易被风吹

Gravity Scale：
  1.0  ：正常重力
  0.3  ：轻飘感（仙侠飘带）
  0.05 ：太空失重感
  1.5  ：沉重拖拽感
```

## 三、求解器参数

```
Solver Settings（在 Chaos Physics Settings 中）：
  Num Iterations：
    约束求解迭代次数
    2  ：最快，布料可能有明显拉伸误差
    8  ：平衡（推荐）
    16 ：精确，开销翻倍
    
  Substep Time：
    每个物理帧的子步长（秒）
    0.016（60Hz）：默认
    0.008（120Hz）：更稳定，开销 ×2
    
  Self Collision Thickness：
    布料自碰撞球半径（cm）
    太小 → 自碰撞失效（布料穿过自身）
    太大 → 布料无法正常折叠
    推荐：衣物厚度的 0.5 倍（约 0.5~1.5cm）

常见布料参数速查表：
  类型        拉伸   弯曲   质量(kg)  阻尼   迭代
  丝绸         0.6   0.02   0.005    0.02    8
  棉布         0.8   0.1    0.02     0.03    8
  厚棉夹克     0.9   0.4    0.1      0.05   12
  皮革         0.95  0.7    0.2      0.06   12
  帆布         1.0   0.9    0.5      0.04    8
  旗帜（丝绸） 0.7   0.05   0.008    0.02    8
```
